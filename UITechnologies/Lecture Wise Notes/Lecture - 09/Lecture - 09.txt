	    (c) Meta Element:
	    -----------------
		- Meta stands for "meta-data".
		- Usually meta data comprises of information about your page, provided to Web Crawlers and Web Spiders.
		- Web Crawlers and Web Spiders.
		- SEO [ Search Engine Optimization ]
		- Meta is one of the option for making page SEO freindly and responsive.
		  [ Responsive page fits according to device and browser ]

		syntax:				UTF : Unicode Transformation Format			8  Bit => for English [us, gb, in]
		    <meta charset = "utf-8">	It is used by BOTS to know the language used in page	16 Bit => for Korean, Japanese etc.

			Difference between <html lang = "en-in"> and <meta charset = "utf-8">
			~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
			<html lang="en-in"     -> Google Engine
			<meta charset="utf-8"> -> SEO
		    				

		- through meta we also request headers
		  syntax:
		      <meta http-equiv = "refresh" content = "4">
			  content in seconds

		Meta viewport
		--------------
		- It makes the page responsive.
		- It is one of the option for making page responsive.
		  responsive means, page will fit content according to device width.

		syntax:
		    <meta name = "viewport" content = "width=device-width, initial-scale = "1">
			
			viewport	: It identify the device screen size & orientation
		    	content		: It refer to content in page which will be equal to device width.
		    	intial-scale	: It is zoom % of content in browser. Default is 100%.
						0.9  = 90%
						1    = 100%
						1.5  = 150%
